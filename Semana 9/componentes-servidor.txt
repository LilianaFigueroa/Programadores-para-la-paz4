Sistema operativo: Este el programa principal que permite que el 
computador funcione de manera adecuada y activa. Se encarga de administrar 
los recursos del equipo (hardware) y de permitir el uso de otras aplicaciones 
para que el usuario pueda interactuar con el dispositivo.


Node.js: Es una herramienta de entonro que permite ejecutar códigos de JavaScript fuera del navegador. 
Se utiliza para crear y poner en funcionamiento aplicaciones y servidores dentro de la terminal.


Aplicación server.js: Es el archivo donde se encuentra el codigo clave y y principal del servidor.
En este archivo se configura las funciones y respuestas que tendra la función del backend de la aplicación
desde conts, if, arrays, entre otros.


Puerto: Es el punto de comunicación que utiliza un servidor o una aplicación para transmitir 
y recibir información a través de una red. Gracias a él, los usuarios pueden acceder a una API o servicio determinado.


Logs: Son los registros de lo que ocurre o se realiza en una aplicación. 
Guardan información útil para detectar errores o problemas en el funcionamiento del sistema. Además, 
pueden contener datos sensisbles que requieren ser protegidos.
